This webapp will generate a Strava API OAuth token for use in some Strava tools.

Security and privacy

Your client ID and secret will be temporarily stored on the server while the auth flow completes. They are purged on successful token generation or after 1 hour, whichever comes first. The generated access token is not stored or used in any way.

Full source code and setup instructions for running your own instance are provided on the project page if this isn't acceptable.
